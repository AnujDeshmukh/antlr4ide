Javav4

A Java 1.7 grammar for ANTLR v4 derived from ANTLR v3 Java grammar. Uses ANTLR v4's left-recursive expression notation. It parses ECJ, Netbeans, JDK etc... Sam Harwell cleaned this up significantly and updated to 1.7! You can test with $ antlr4 Java.g4 $ javac *.java $ grun Java compilationUnit *.java

  • compilationUnit Top

    starting point for parsing a java file

    Text notation:

    compilationUnit : packageDeclaration? importDeclaration* typeDeclaration* EOF ;
